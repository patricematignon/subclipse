Install Subclipse 1.10.x

See the changelog for the 1.10.x release of Subclipse. Unless otherwise noted, the best available version is always the most recently released version.

Detailed installation instructions for Subclipse in Eclipse 3.x can be found on the page Installation Instructions. Generally speaking, Subclipse is installed and updated from within Eclipse. There is no specific external download needed.
