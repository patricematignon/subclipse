Connecting to a Repository

Procedure

To connect to a repository you will need to open the SVN Repository Exploring Perspective.


It is in this perspective that you can create connections to various repositories, and manipulate remote resources for tagging and branching.

If you already are using SVN

If you are already working with a working copy, you still need to let Eclipse know that the SVN working-copy is already in place. See Connecting an Existing Project for more information.
