Mark As Deleted

This function is initiated via the Team > Mark As Deleted context menu.

Overview

Use this command to identify a missing resource as to be deleted to SVN.

Process

Select the missing resource and choose Mark As Deleted from the Team context menu. This option is only available in the Pending SVN Operations view. This is due to the fact that the file does not exist, so it is not possible to select it in another view to perform an action on it. Missing resources will also appear in the Synchronize View and the Commit Dialog and can be deleted on the fly as part of a commit.
