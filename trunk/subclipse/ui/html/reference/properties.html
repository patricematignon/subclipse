Subversion Resource Properties Page

All resources in a project managed by Subversion have one additional Eclipse property page as shown when selecting "Properties" on the resource. The property page appears like this:

The information available corresponds to the information given by 'svn info' on the Subversion command line client.

Please note: This Property page is not to be confused with Subversion Properties which are versioned tag-value pairs stored in the repository, and edited in Subclipse in the SVN Properties View.
