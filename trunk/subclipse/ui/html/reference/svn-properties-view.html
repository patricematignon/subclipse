SVN Properties View

In the SVN Properties View, you can create, edit, add, remove Subversion properties for the selected resource.

The properties are versioned data themselves - if you change a property, you need to commit it before other developers can see it.

Each property for the selected resource is listed in the view, and edited by using the context menu, which also includes the Set Keywords command (also available in the Team menu).

In general, properties are just name/value pairs that mean something to the specific usage of the repository. However, some special properties have explicit meaning to Subversion itself, to the Subversion client, or to UI agents like Subclipse.

Reference

Team > Add to svn:ignore.

Issuetracker support in Subclipse.
