A Quick Start Guide and Reference

The Getting Started section will take you through the basics of the SVN plug-in, such as...
  • Creating a repository
  • Connecting to a repository location
  • Importing a new project into the repository
  • Checking out a project from a repository into your workspace
The section on Daily Work focuses on the day-to-day work cycle:
  • Editing
  • Updating
  • Committing your work
  • Synchronizing with the repository (using the Synchronize View)

The Reference section will take you through each feature the plug-in provides.

The FAQ section provides some quick answers to some of the common questions

However, if you are already an SVN user, then the feature-by-feature comparison of the SVN command line and SVN plug-in may be your best starting point.
