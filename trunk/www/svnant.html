Subclipse | SvnAnt | SvnClientAdapter

SvnAnt is an ant task that provide an interface to Subversion revision control system that is a compelling replacement for CVS in the open source community.

Svn uses javahl (a java interface for the subversion api) if it can find the corresponding library (svnjavahl.dll on windows). Otherwise it uses experimental svn command line interface.

SvnAnt supports most of the major Subversion commands.

Documentation for svnant is available from our repository. Use username and password of "guest" if prompted. http://subclipse.tigris.org/svn/subclipse/branches/svnant/1.0.x/doc/svn.html.
