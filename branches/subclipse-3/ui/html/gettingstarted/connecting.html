Connecting to a Subversion Repository

To connect to a Subversion repository you will need to open the SVN Repository Exploring Perspective.


It is in this perspective that you can create connections to various Subversion Repositories, and manipulate remote resources for tagging and branching.

If you already are using Subversion

If you are already working with a working copy, you still need to Eclipse know that the Subversion working-copy housekeeping is already in place. You can do this by select the project you want to import and right click to select Team > Share Project....
