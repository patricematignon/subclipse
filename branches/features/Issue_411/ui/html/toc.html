A Quick Start Guide and Reference

The Getting Started section will take you through the basics of the Subclipse Eclipse plugin for Subversion such as...
  • Creating a Repository
  • Connecting Eclipse to the repository or using an existing workspace
  • Importing a new project into the respoitory
  • Checking out a project from a repository into your Eclipse workspace
The section on Daily Work focuses on the day-to-day work cycle:
  • Editing
  • Updating
  • Committing your work
  • Synchronizing with the repository (using the Synchronize View)

The Reference section will take you through each feature the plug-in provides.

The FAQ section provides some quick answers to some of the common questions

However, if you are already a Subversion user, then the feature-by-feature comparison of the Subversion client (command line) and Subclipse may give you the best starting point.
